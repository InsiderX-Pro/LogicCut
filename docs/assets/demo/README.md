# Demo Assets

These lightweight assets are used by the public-preview README. They are intentionally small and do not include private cookies, API keys, model weights, or generated long-form media.

| File | Purpose |
| --- | --- |
| `logiccut-demo-hero.jpg` | README hero collage showing the repurposing workflow. |
| `theme-opener-demo.gif` | Short visual preview of the theme opener output. |
| `theme_opener.jpg` | Demo Gallery preview for theme opener. |
| `comment_item.jpg` | Demo Gallery preview for public comment recap. |
| `translation.jpg` | Demo Gallery preview for localized subtitles. |
| `templates.jpg` | Demo Gallery preview for HTML/card-style overlays. |

Release checks:

- Keep GIFs under 10 MB.
- Do not include usernames, cookies, API keys, tokens, or local machine paths.
- Prefer cropped previews over shipping generated `.mp4` files.
- Keep previews aligned with implemented features and current README wording.
